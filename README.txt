ce referentiel git est dédié à l'apprentissage de GIT
c'est une sorte de projet "sandbox" pour s'exercer à effectuer des opérations en équipe
-------------
par exemple une équipe pourra effectuer des modifications sur la partie 
basicHtmlCssJs ou basicJava ou basicNodeJs
tandis qu'une autre équipe pourra effectuer des modifications sur la partie 
basicHtmlCssJs2 ou basicJava2 ou basicNodeJs2
